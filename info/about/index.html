Hello! My name is Igor. I'm studying A Course in Miracles and sometimes I write notes about it.

I had been using a new tool http://hypothes.is which allows writing notes on any web page on the Internet, but as the number of notes increased I decided to build this website to have access to all of the notes from one place.

Also I couldn't find a source of random quotes from the course to read at the end of study groups sessions for every member, so called ACIM Fortune Cookies.

As a result of all this, on this website you can read my notes about the Course and you can get ACIM Fortune Cookies for all of your study group's members, enjoy ;)

Since 2017 I started doing online regression sessions for those who are willing to experience ACIM's teaching in practice. Please visit the Healing page to learn more about that.
